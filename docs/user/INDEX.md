# 用户手册索引

本目录包含 Nacos Gateway 的用户手册，面向网关用户和运维人员。

## 目录定位

面向**网关用户**和**运维人员**，提供安装、配置、部署和运维的实用指南。

## 基本信息

| 文档类型 | 数量 | 说明 |
|----------|------|------|
| 安装指南 | 1 | 环境要求、构建、运行 |
| 配置指南 | 1 | 配置结构、限流、负载均衡 |
| 部署指南 | 1 | systemd、Docker、Kubernetes 部署 |
| 监控指南 | 1 | 健康检查、日志、常见问题 |
| API 参考 | 1 | 管理接口、配置协议 |

## 快速开始

1. 查看 [安装指南](installation.md) 了解环境要求
2. 参考 [配置指南](configuration.md) 配置网关
3. 阅读 [部署指南](deployment.md) 部署服务
4. 使用 [监控与故障排查](monitoring.md) 解决问题

## 文档维护

### 更新时机

- 版本发布时更新安装和部署指南
- 新增配置项时更新配置指南
- 发现新的常见问题时更新监控指南

### 注意事项

- 配置示例应与最新代码版本一致
- 部署脚本应经过实际测试验证
