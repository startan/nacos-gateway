# 架构决策记录索引 (ADR)

本目录包含 Nacos Gateway 项目的架构决策记录（Architecture Decision Records）。

## 目录定位

面向**项目历史追溯**，记录重要的架构决策及其背景和后果。

## 基本信息

| ADR 数量 | 状态 |
|----------|------|
| 3 | 全部已接受 |

ADR 是记录重要架构决策的文档，包括：
- 决策的背景和上下文
- 做出的决策及其原因
- 决策的后果（正面和负面）
- 考虑过的替代方案

## 创建新 ADR

1. 复制 `template.md` 为 `NNN-title.md`（NNN 为序号）
2. 填写各章节内容
3. 更新本文件的索引表
4. 提交代码审查

## 文档维护

### 更新时机

- 做出重要架构决策时创建新 ADR
- 决策内容变更时更新对应 ADR

### 注意事项

- ADR 应记录决策时的背景和思考过程
- 一旦决策做出，ADR 内容不应再修改（可标记状态为"已废弃"）
- ADR 编号应连续，不重复
